How to install
==============

1. Get the ooz source code (https://github.com/powzix/ooz) and unpack to this directory.
2. Remove testdata directory (it's not needed, just wastes disk space).
3. Patch it with ooz.patch with use of GNU patch tool (could be found in git installation):
   patch -i ooz.patch
4. Recompile UE Viewer.
